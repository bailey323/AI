Add User

You can add a new user via the Client Maintenance tab

To add a Client User

  1. Log onto C Series Admin as a valid administrator user
  2. With the Client Maintenance tab in focus, select Setup - Client User, and click Add
  3. On the Client User screen, in the User Group field, click the cllipsis, and select a Client User Group to associate with this Client User.
  4. Type an appropriate User ID, Username and Password.
  5. Enter Users email address
  6. Select correct Locale and Time Zone
  7. Choose correct Data Role for Client User
  8. On the bottom of the screen, click Add.

You now need to add Function Roles.

To add Function Roles

  1. On the User Function Roles screen, click Add Function Roles to assign roles to your user. The Add Function Role filter screen appears
  2. On the Add Function Role filer screen, highlight the role or roles you wish to assign to your user.
  3. Click the single right arrow to select one role and the double right arrow to select all roles
  4. Once you have finished selecting roles, at the top of the screen click Save. A dialog box appears
  5. In the dialog box, click OK. The Add User screen appears with the Funtion Roles added at the bottom

On the bottom of the screen, click Update

You have successfully added a client user.

When Client Users are added, their status will be "EN" (Entered). Once a Client User has been added, you must approve the addition.

Password Reset

Once a Client User is added, you can reconfigure that User, as necessary.

Note: You cannot change a Client User's User Group or User ID once they have been established

  1. Log onto C Series Admin as a valid administrator user
  2. With the Client Maintenance tab in focus, select Setup - Client User, and click Modify.
  3. On the filter screen, select filter criteria, and/or clickEnter.
  4. Select the User Group you want to work with
  5. On the Client User screen, modify any information as needed (eg. reset password)
  6. If you want to assisgn roles to that User, click Roles.
  7. Click Update and OK on the confirmation prompt.

    8. You have successfully modified the client user.

Note:

  • Fields shaded light blue are mandatory.
  • Click Reset to clear the details that were entered in the Client user screen
  • Click Back to select another User

Maintain Users

Follow the User Search process and then maintain the neccessary fields. These include:

  • General contact details/security information
  • Role cannot be maintained. If someone needs to change roles, you need to submit a request to Barclaycard Support (Technical Helpdesk) and they will arrange the amendment for you
  • Disable/Re-enable to prevent a user from accessing the system, check them as Disabled.
  • Locked out/Unlock - a user can be manually locked out by an administrator or through three invalid access attempts. Users can be unlocked by an administrator or by your Barclaycard support team.

Note: for security reasons, users are not told they are locked out, only that their login was unsuccessful. Lockout occurs after 3 unsuccessful login attempts.

Create User

  1. From 'Administration' section, select 'Manage Users'.
  2. Select 'Create User'
  3. Enter the details for your user (name, email address, contact numbers, postal address, etc), create a User ID, set a payment authorisation limit if required then hit 'Continue'
  4. You can assignroles by checking the box next to the relevant role. When you've finished, hit 'Continue'
  5. Depending on the role(s) you assign, you may need to authorise the changes using your Smartcard and card reader.
  6. Finally, hit 'Confirm New User' and tell the new user to expect an email containing their activation code. For security reasons, we send Smartcards and card readers to the administrator who created the new user. They should arrive within five business days of the email
  7. Once the user receives their activation code, they'll need to use this to login along with their 'Customer ID' and 'User ID'. The customer ID is the same one used by every user on your Bankline profile and the user ID will be the one you've created for them (as per the steps above). You'll need to share both these IDs with them.

Authorise Change

For any action subject to 'Dual administration', a second user needs to review the changes that the first user has made and either approve or reject them.

To do this:

  1. Go to the 'Authorise change' screen, which shows any pending changes you can authorise
  2. Under the heading 'Details' select the change you want to approve or reject
  3. From here you can select to either accept or reject the changes
  4. If this change requires a Smartcard, you'll be prompted to use it

Handy tip - You can't view or approve any changes you've made yourself.

FAQ

How do I get a replacement card reader or Smartcard?

To order a Smartcard reader:

  1. From 'Administration' section, select 'Manage Users'
  2. Select 'Order Smartcard reader'
  3. Hit 'Continue', then 'Confirm'

To order a replacement Smartcard:

  1. From 'Administration' section, select 'Manage Users'
  2. Select relevant User ID
  3. Scroll to the bottom and select 'Edit roles/Smartcard'
  4. Now hit 'Order replacement Smartcard'
  5. Select the address you'd like it sent to
  6. If you want to order a Smartcard reader at the same time, check the 'New Smartcard reader required' box
  7. Finally, hit 'Continue' then 'Confirm'

Reminder: if you have Dual administration switched on, you'll need a second administrator to authorise your order

My user has got their login details wrong and is now locked out of Bankline, how do I re-activate them?

To help them regain access you can use the following options:

  1. If the user still remembers their PIN and password, go to 'Manage Users'
  2. Select the User ID and then hit 'Re-instate User'
  3. If they don't remember their PIN and password, from 'Manage Users' select the 'User ID'
  4. Select 'Order Activation Code'

How do I re-instate a user who entered their PIN or password incorrectly?

If a user has entered their PIN or password incorrectly, but they know the correct details, an administrator can re-instate them.

  1. From 'Administration' select 'Manage Users'
  2. Click on the relevant User ID
  3. Select 'Re-instate User'. The user will now be able to log in to the system using their exisiting PIN and password

Handy hint: the user status on the 'Manage Users' screen needs to be disabled for the option to be available.

How do I view, edit, delete or suspend users?

  1. From 'Administration', select 'Manage Users'
  2. Select the relevant User ID
  3. 3.This opens the 'Edit User' screen. From here you can view and edit the user's details and click through to edit roles, order a new activation code, and suspend or delete the user.

If you have any further issues please visit: Admin Help Page

  • Click on Create User
  • Enter the user email address and click on Verify Email
  • If the e-mail address already exists within Connexis Cash, you will need to confirm whether you want to grant the user access to this new entity. The entity will be added to the users existing profile


    • Fill in the user and token activation information on the new screen

  • Enter the first and last name, and confirm the email address
  • Choose the type of user token: mobile application or physical token<



  • If the mobile application is chosen, enter a valid mobile (recommended) or a landline phone number. The new user will receive an SMS/landline call with a code to launch the activation process.
  • If the physical token is selected, choose a token activation method (digital or sealed letter) and enter a phone number and a mailing address.
  • For a digital process (recommended), enter a mobile number for receiving an SMS.
  • If there is no mobile number, enter a landline number for receiving a token activation code via a call from the bank.
  • For a sealed letter process, enter a mailing address. A paper sealed letter will be sent to the physical address.


  • Click on Create to submit the request

Managing user security access right

The Users menu displays all the registered users along with their details in Administration > User management > Users.

Navigate to the Users details screen by double-clicking on a user

  • Identification: Review user information and inform the bank representatives of any changes
  • Security Access: Define and manage the user's security and access rights.
  • Profiles: Assign profiles to the users


    • Assign or modify access rights to user(s) from Users menu.

    By default, users are set as Watchers with no permissions.

    Assigning user rights

  • Double-click on a user
  • Review data in the Identification tab.
  • Click Submit if any new information has been entered.

    • Assigning user rights

  • Click on the Security access tab.
  • Select a user type from: Watcher/Initiator/Authorizer/Sender/Checker
  • For authorizers, assign an authorization group. The authorization groups can be defined in detail in the Authorization menu. Payment permissions will be displayed corresponding to the selected user type.
  • Choose the payment rights:
    • View payroll - View the payroll details
    • Delete or restore - Delete or restore payments
    • Cancel - Cancel payments
    • Create a counterparty - Create and savbeneficiaries/drawees
    • Mobile accessibility - Access on a mobile
    • Same security access as - Copy security access rights from another user. Further modifications can be performed without modifying the original user’s rights
    • Authorization level - Applicable only for the simple payment authorization mode. An authorizer can sign (single) or may require a second authorizer (double)
    • FX interactive - Ability to book an FX rate
    • Simplified authorization - Simpler authorization process is possible
    • Cannot sign own payment - Restrict the authorizer from signing own payment
    • Cannot sign own payment alone - Restrict the authorizer from signing own payment on their own, but allow signing with other authorizers
    • Allow mobile authorization - Authorize payments from a mobile
    • Allow once-off beneficiary/drawee - Create and use a beneficiary/drawee without saving it to the database


  • Click on Submit to assign rights or Reset to revert to the previous settings

User Management

All user maintenance will be completed in iPortal.

To access the User screen, navigate to Admin. This defaults to the 'User' menu option. Within this screen you can:

  • Add/amend/delete a user
  • Amend the roles assigned to a user
  • Change a user's status.

To add a new user, simply click 'Add New User' from within the User screen.


To amend a user, click the hamburger menu on the user card. A list of actions will appear, simply select the action you wish to take.


Please note: Actions that aren't applicable will be greyed out. For example, if a user is active, the Activate User option will be grey.
